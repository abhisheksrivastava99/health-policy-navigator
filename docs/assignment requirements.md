# MH6818 FinTech Innovation with AI — Individual Assignment 2 Requirements

## Overview
This assignment requires designing an **agentic AI solution** for a chosen area of financial services.

The objective is to show how LLM agents can improve or transform an existing workflow while ensuring:
- appropriate governance,
- measurable economic value.

## Scope Selection
Choose **one** focus area from:
- Insurance
- Investment research

For the selected area, explain:
1. The current workflow or process.
2. The key challenges or inefficiencies.
3. The motivation for innovation.
4. Why an agentic AI approach is appropriate.

## Proposed Agentic System Architecture
Describe the architecture of the proposed agentic system, including:
1. The different agents involved, for example:
   - Planner agent
   - Research agent
   - Execution agent
2. How these agents interact and coordinate.
3. Integration with tools, databases, or external systems.
4. How context and memory are managed.

### Required diagram
- Include a **system architecture diagram**.

## Agent Reasoning and Human Oversight
Explain:
1. How each agent reasons.
2. How each agent performs tasks.
3. How each agent makes decisions.
4. Where **human-in-the-loop** is required.

### Bonus marks
- Students who develop a prototype using tools such as **LangGraph**, **Google ADK**, or **Dify** can receive **up to 10 additional marks** for this component.

## Data Requirements
Describe the data needed for the proposed system, including:
1. Data sources.
2. How the data will be prepared.
3. How agents retrieve, access, or use the data.

## Risk, Governance, and Monitoring
Assess the potential risks associated with the proposed solution and propose:
- appropriate governance mechanisms,
- monitoring approaches.

## Economic Value Measurement
Propose a methodology to measure the **economic value** created by the proposed solution.

## Assumptions
- Clearly state all assumptions where needed.

## Expected Quality of Submission
The submission should:
- demonstrate innovative thinking,
- present a technically sound agentic architecture,
- connect AI capabilities to business value creation,
- identify relevant governance and risk considerations.

## Assessment Criteria
The assignment is evaluated out of **100 marks** as follows:

| Criterion | Weight |
|---|---:|
| Selection of focus area and problem statement | 10% |
| Architecture design of the proposed agentic system | 25% |
| Explanation of agent reasoning, task execution, and decision logic | 25% |
| Data preparation and description | 20% |
| Risk, challenges and governance considerations | 10% |
| Measurement of economic value | 10% |
